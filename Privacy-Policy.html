Java Lava Bot Privacy Policy

Terms updated: 10/26/2025 — Effective: 11/1/2025

1. Info We Collect

We collect the following Information when you use Java Lava:

  • Server Information: Server name, member count, and server ID.
  • Command Usage: Commands used and their parameters.
  • Error Logs: Information about errors encountered while using the bot.

2. Data Storage & Security

We store data securely and comply with Discord's Developer Policies. Data is retained only as long as necessary for the bot's functionality.

3. Data & Privacy

Java Lava will NEVER sell, trade, or share your personal information with third parties. Data is used solely to improve bot performance and user experience.

4. Data Retention

Java lava's data is kept for 2 to 4 week periods, after which it is automatically deleted. Users can request data deletion by contacting the developer's dms via Discord.

5. Updates to this Privacy Policy

These terms may and WILL change as the bot evolves. Notifications will be posted in our support server's #bot-updates channel.

6. Contact Information
